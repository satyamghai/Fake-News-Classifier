# 📰 Fake News Classifier

## 📌 Overview

A machine learning web application that classifies news articles as **Fake** or **Real** using Natural Language Processing (NLP) techniques and supervised learning algorithms.

The project includes data preprocessing, feature engineering, model training, evaluation, and deployment using Streamlit.

## 🚀 Live Demo

Streamlit App: https://fake-news-classifier-gwrvzo9r88uiat8optxvuk.streamlit.app/

## 🛠️ Technologies Used

* Python
* Pandas
* NumPy
* Scikit-Learn
* NLTK
* Streamlit
* Matplotlib
* Seaborn

## 📁 Dataset

Source: Kaggle Fake & Real News Dataset

Files:

* Fake.csv
* True.csv

## 🔍 Project Workflow

1. Data Cleaning & Preprocessing
2. Text Vectorization using TF-IDF
3. Model Training & Evaluation
4. Model Comparison
5. Streamlit Web Application Deployment

## 🤖 Models Evaluated

* Naive Bayes
* Logistic Regression
* Support Vector Machine (SVM)
* Random Forest
* Decision Tree

## 📈 Results

* Multiple machine learning models were evaluated and compared.
* Best-performing models achieved approximately 99% accuracy on the test dataset.

## ▶️ How to Run

Install dependencies:

pip install -r requirements.txt

Run the application:

streamlit run app.py
